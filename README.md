
# Cannabis Legislature Tracker

The Cannabis Legislature Tracker is an open-source tool that monitors and aggregates cannabis-related bills from all 50 U.S. states, starting from January 1, 2024 through the present day. It provides a simple, scannable interface for users to browse and compare legislative activity across the country.

## Features

- Automated daily scraping of state legislative websites
- Plain English summaries of each bill's key provisions and potential impacts
- Direct links to official bill text and tracking pages
- Filterable by state, status, topic, and more
- Searchable by keyword
- Mobile-friendly responsive design

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Python 3.6+
- Beautiful Soup 4
- Requests

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/cannabis-legislature-tracker.git
   ```
2. Install required packages
   ```sh
   pip install -r requirements.txt
   ```
3. Run the scraper
   ```sh
   python scraper.py
   ```
4. Open index.html in your browser
